Detailed Description

Simple ThreadPool that creates threadCount threads upon its creation, and pulls from a queue to get new jobs.

This class requires a number of c++11 features be present in your compiler.

void ThreadPool::AddJob ( const std::function< void()> &  job)

Add a new job to the pool. If there are no jobs in the queue, a thread is woken up to take the job. If all threads are busy, the job is added to the end of the queue.

◆ JoinAll()

void ThreadPool::JoinAll ( )
inline

Join with all threads. Block until all threads have completed. The queue may be filled after this call, but the threads will be done. After invoking ThreadPool::JoinAll, the pool can no longer be used.

◆ WaitAll()

Wait for the pool to empty before continuing. This does not call std::thread::join, it only waits until all jobs have finished executing.
