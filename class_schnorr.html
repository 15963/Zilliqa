const unsigned int Schnorr::PUBKEY_COMPRESSED_SIZE_BYTES = 33
static

Public key is a point (x, y) on the curve. Each coordinate requires 32 bytes. In its compressed form it suffices to store the x co-ordinate and the sign for y. Hence a total of 33 bytes.
